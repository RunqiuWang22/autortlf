# Sample Output Tables

This directory contains sample RTF (Rich Text Format) tables generated by AutoRTLF for demonstration purposes.

## Available Sample Tables

### Baseline Characteristics
- **`baseline0char0itt.rtf`** - Intent-to-Treat population baseline demographics
- **`baseline0char0itt0overall.rtf`** - Overall baseline characteristics summary
- **`baseline0char0white.rtf`** - White population baseline characteristics

### Adverse Events
- **`ae0specific0des0sae01.rtf`** - Serious adverse events (≥1% threshold)
- **`ae0specific0des0sae05.rtf`** - Serious adverse events (≥5% threshold)
- **`ae0specific0soc05.rtf`** - Adverse events by System Organ Class (≥5% threshold)
- **`ae0specific0soc050overall.rtf`** - Overall adverse events summary

### Combined Output
- **`combined/X99-ia01_combined_tlf_20251014.rtf`** - All tables combined into single document

## File Formats

- **RTF**: Rich Text Format for Microsoft Word compatibility
- **Publication-ready**: Properly formatted for clinical study reports
- **FDA-compliant**: Standardized formatting for regulatory submissions

## Usage

These sample tables demonstrate:
- AutoRTLF output quality and formatting
- Standard clinical trial table structures
- RTF formatting capabilities
- Batch processing results

**Note**: These are demonstration outputs using sample data. For production use, replace with your actual study data and regenerate tables.
